Agile Workshops

In these group workshops I act as a facilitator. The applied methodologies are Design Thinking, Sprints, Lean Startup and Scrum.

PortS Mobile Product Portfolio App

The PortS mobile product portfolio app was created in order to improve communication between sales people and product managers at SPIRIANT. It allows the sales managers to have the SPIRIANT product portfolio available at any time without the need of creating powerpoint presentations for customer meetings.

"Ap(p)prende" Vocabulary App UX Design

While researching and designing "Ap(p)rende", an app to learn new vocabulary, I learned the most important terms, tools and techniques in the field of user experience design. Starting from competitor research and user research, I defioned the user needs and created personas. The next step was to create user flows and task analyses. Wireframes and prototypes were used for usability testing after that.

"Homecourt" Streetball App UX Design

Homecourt is an application that allow streetballers to find courts, learn about these courts and team up with other ballers in order to go thhere for a session of casual pick-up basketball. The project included tasks like user research, personas creation, user journey mapping, user flow mapping, the creation of low-, mid- and high-fidelity prototypes and user testing.
